# G2i Dashboard Enhancement

This project was created to showcase the use of filters in the G2i job dashboard. This job board is displayed as the main hub for developers to apply for jobs in the platform. Including filters enhance the experience as well as excluding the need of having separate routes for each refined search based on roles / categories of work.

## Background

- Inspired on data from their prod API, but redacted a few names to preserve privacy.

## Technologies used

- `pnpm`
- React
- TypeScript
- Vite
- [Mantine](https://mantine.dev/)
- Frontend only, in-memory filtering.
- QuickTime for video recording.
